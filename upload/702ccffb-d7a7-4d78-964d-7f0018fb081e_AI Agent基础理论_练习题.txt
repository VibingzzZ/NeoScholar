### 二、简答题

**1. 请简述反应型 Agent 与慎思型 Agent 的主要区别，并分别举例说明。**

**参考答案：**

*   **反应型 Agent：** 遵循“感知-行动”的直接映射，不依赖内部状态或世界模型。它根据当前感知到的环境信息，立即执行预定义的动作。其优点是响应速度快，结构简单；缺点是缺乏灵活性，无法处理复杂或需要历史信息的情况。
    *   **举例：** 一个简单的扫地机器人，当它前方的传感器检测到墙壁时，立即执行“转向”动作。
*   **慎思型 Agent：** 包含一个显式的世界模型和推理引擎。它会感知环境，更新内部模型，基于模型进行推理和规划，然后选择最佳行动。其优点是能处理复杂、长期的任务，更具灵活性；缺点是计算开销大，响应速度可能较慢。
    *   **举例：** 一个智能导航系统，它会接收当前位置和目标地点（感知），利用地图数据（世界模型）规划出一条最优路径（推理/规划），然后给出导航指令（行动）。

**2. 解释“感知-行动循环”在 AI Agent 设计中的重要性，并描述其核心步骤。**

**参考答案：**

“感知-行动循环”是智能 Agent 与外部环境交互的基本框架，它体现了 Agent 的认知过程。其重要性在于，它提供了一种通用的、可循环的机制，使 Agent 能够动态地响应环境变化，并通过行动影响环境，从而达成目标。

其核心步骤包括：
1.  **感知 (Sense/Perceive):** Agent 通过传感器从环境中获取信息。
2.  **思考/规划 (Think/Plan):** Agent 处理感知到的信息，结合内部知识或模型，决定下一步要采取的行动。这一步可能涉及推理、规划、学习等过程。
3.  **行动 (Act):** Agent 通过执行器对环境施加影响，执行决策好的动作。

循环往复，Agent 持续地感知环境变化，做出决策，并采取行动，形成闭环。

**3. 与单个 Agent 相比，多 Agent 系统在解决复杂问题时具有哪些优势？请至少列出三点。**

**参考答案：**

与单个 Agent 相比，多 Agent 系统的主要优势包括：
1.  **效率提升：** 多个 Agent 可以并行工作，同时处理不同部分的任务，从而大大加快问题解决的速度。
2.  **鲁棒性增强：** 系统中的某个 Agent 出现故障时，其他 Agent 可以接替其工作或调整任务分配，使系统仍能正常运行，避免了单点故障问题。
3.  **能力扩展：** 可以通过组合具有不同专长或资源的 Agent（异质 Agent）来解决单个 Agent 无法胜任的复杂、跨领域问题。例如，一个由搜索、计算、翻译 Agent 组成的系统可以完成复杂的信息检索任务。
4.  **灵活性：** 多 Agent 系统可以动态地适应环境变化和任务需求，例如根据任务紧急程度动态分配资源，或引入新的 Agent 来处理新出现的问题。
